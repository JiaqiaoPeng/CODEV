 Code Linter
======================================

##使用前提
目前软件依赖node.js, npm以及standard包

* 安装node
https://nodejs.org/en/download/

* 安装js linter
```bash
$ npm install standard --global
```


##使用方法

1. 选择需要整理的文件夹(非递归)

2. 选择输出文件夹

3. 点Standardize the code

##注意事项

**沙盒模式已经关闭,而且没有采取文件保护手段,所以目标文件夹的文件安全并不能保障.**
